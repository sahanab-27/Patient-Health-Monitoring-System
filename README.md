# Patient Health Monitoring System

This project is built using Arduino to monitor patient health conditions.

## Components Used
- Arduino Board
- Heart Rate Sensor
- Temperature Sensor

## Future Improvements
- Mobile app integration
- Cloud data storage
- Alerts for abnormal health conditions

## Technologies Used
- Arduino (.ino)
- Sensors

## How it works
The system collects patient health data using sensors and processes it using Arduino.
